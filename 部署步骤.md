# 线上 Word 导出 · 部署步骤（Render 免费版）

这个小服务让**线上题库**也能导出 Word（与本地完全相同的效果）。只需部署一次。

## 一、把这个文件夹放到 GitHub

1. 注册/登录 https://github.com ，点右上角 **+ → New repository**，名字随意（如 `math-bank-word`），建好。
2. 把本文件夹 `word-export-server` 里的 4 个文件上传：`app.py`、`导出Word.py`、`requirements.txt`、`render.yaml`
   （在仓库页面点 **Add file → Upload files**，把这几个文件拖进去，提交即可。）

## 二、在 Render 部署

1. 注册/登录 https://render.com （可用 GitHub 账号一键登录）。
2. 点 **New + → Web Service**，选择刚才那个 GitHub 仓库。
3. 关键设置（若没自动带出就手动填）：
   - **Language/Runtime**：Python 3
   - **Build Command**：`pip install -r requirements.txt`
   - **Start Command**：`uvicorn app:app --host 0.0.0.0 --port $PORT`
   - **Instance Type**：Free
4. 展开 **Advanced → Environment Variables**，加两条：
   - `WORD_EXPORT_TOKEN` = 你自己定一个口令（如 `mymath2026`，别人不知道就调不了）
   - `PAGES_BASE` = `https://math-bank.pages.dev` （你的题库图床地址，一般就是这个）
5. 点 **Create Web Service**，等几分钟构建完成。
6. 顶部会出现服务网址，形如 `https://math-bank-word.onrender.com`。
   打开它，看到 `{"ok":true,...}` 就表示成功了。

## 三、把网址和口令告诉我

把这两样发我：
- 服务网址（`https://....onrender.com`）
- 你设的口令（`WORD_EXPORT_TOKEN` 的值）

我会填进网页（`index.html` 顶部的 `WORD_CLOUD_URL` 和 `WORD_CLOUD_TOKEN`），再生成部署版。之后你在**线上**点「导出 Word」就会下载一个 zip（里面是空白卷 + 含答案两份）。

## 说明
- **免费档会休眠**：长时间没用后，第一次导出要等约 30–60 秒（服务在唤醒），之后就快了。
- 本地工作台的 Word 导出**不受影响**，仍是存到「导出Word」文件夹并自动打开。
- 口令写在网页里（前端可见），它的作用是挡掉随手的滥用；要更强保护可改用别的方式，按需再说。
